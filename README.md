# C-RPG

A text based RPG game made in C

## Prerequisites

To compile the project you need [Meson](https://mesonbuild.com/SimpleStart.html) and `Make` in your system.

## Building & Running

To build the project you must first run `make build` which will generate the `build/` directory for meson.
You can run `make run` which will automatically compile and run the game.
